# Privacy Policy

Tab Mission Control does not collect, sell, transfer, sync, or transmit personal data to the developer or any third party.

## What the extension accesses

- Open tabs, tab titles, tab URLs, and browser window information
- Recently closed tab information provided by the browser `sessions` API
- A single local theme preference stored with extension storage

## How that data is used

- To show the Mission Control tab overview
- To activate, move, close, hibernate, and restore tabs at the user's request
- To remember the selected `system`, `light`, or `dark` theme on the current browser profile

## What is not collected

- No analytics
- No telemetry
- No crash reporting
- No advertising identifiers
- No remote account or sign-in data
- No cookies or browsing history are sent to the developer
- No data is sold or transferred for purposes unrelated to the extension's single tab-management feature

## Data storage

- Theme preference is stored locally in the browser's extension storage
- Tab and session information stays inside the browser APIs and is not persisted by the extension beyond normal browser behavior

## Network behavior

- The extension does not call developer-operated servers or third-party APIs
- It ships with no remotely hosted code
- It has no host permissions and no content scripts
- In Chrome, favicon rendering uses the browser's local `/_favicon/` resource instead of requesting site favicons directly
- In Firefox, the popup only renders safe local or browser-provided favicon URLs; otherwise it falls back to the bundled icon instead of requesting remote favicon resources

## Chrome Web Store Limited Use Statement

If this extension is published to the Chrome Web Store, the use of information from Chrome permissions and browser APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
